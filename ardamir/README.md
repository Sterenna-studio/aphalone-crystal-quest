# Ardamir — Panneau de réactions

Petite page web statique pour interagir avec les réactions du personnage.

## Lancer en local

Ouvre simplement `index.html` dans ton navigateur.

## Mettre en ligne rapidement

Options simples :
- Netlify Drop : glisse-dépose le dossier complet sur Netlify Drop.
- GitHub Pages : pousse le dossier dans un dépôt GitHub puis active Pages.
- OVH / FTP : envoie `index.html`, `styles.css`, `script.js` et le dossier `assets/` dans un sous-dossier de ton site.

## Modifier les dialogues

Les textes sont dans `script.js`, dans le tableau `emotes`.
